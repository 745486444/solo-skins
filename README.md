# Solo 官方皮肤

## 简介

这里是 [Solo](https://github.com/b3log/solo) 开源博客系统的官方皮肤仓库。

Solo 官方皮肤中有很大一部分是其他博客系统皮肤的仿款，为充分尊重原作者的劳动成果：

* 仿款皮肤完全遵循原作皮肤开源协议
* 仿款皮肤页脚处标明原作地址，非常感谢原作者的创意和分享
* 如果作者不愿意授权，也欢迎联系我们，我们会及时下架

## 皮肤

预览及下载可参见 [Solo 主题](https://solo.b3log.org/#themes)
